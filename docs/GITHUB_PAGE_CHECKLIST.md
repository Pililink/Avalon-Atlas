## Social Preview

Recommended preview image:

- Size: 1280 x 640
- Content: app name, compact assistant UI screenshot, and a short line such as `Albion Online Avalon road map assistant`
- Avoid using official Albion Online branding in a way that implies endorsement.

## Release Page

Each release should include:

- One portable zip asset.
- Short install instructions.
- Notable changes.
- Disclaimer that Avalon Atlas is an unofficial community tool.
- SHA256 checksum when available.
